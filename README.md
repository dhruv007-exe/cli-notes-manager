# 📝 CLI Notes Manager

A lightweight command-line Notes Manager built in **C++** that allows users to create, view, search, and delete notes. The project demonstrates object-oriented programming, file handling, and the Standard Template Library (STL).

---

## ✨ Features

- ➕ Add new notes
- 📋 View all saved notes
- 🔍 Search notes by title or keyword
- 🗑️ Delete notes
- 💾 Automatically save notes to a file
- 📂 Load existing notes on startup

---

## 🛠️ Tech Stack

- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)
- File Handling

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/cli-notes-manager.git
```

### Navigate to the project

```bash
cd cli-notes-manager
```

### Compile

```bash
g++ *.cpp -o NotesManager
```

### Run

**Windows**
```bash
NotesManager.exe
```

**Linux/macOS**
```bash
./NotesManager
```
